I'm a Wife App

A soft place to land before you press send. Name what happened, breathe, remember you're not the problem — then tell him about it like a grown woman instead of a slammed door.

🚀 How to Get Started

  1. Install dependencies: Run npm install in terminal
  2. Start the server: Run npm start in terminal
  3. Open the app: The React app will load below automatically
  4. SMTP is configured: Email credentials are in .env.local
